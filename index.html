LAYOUT

  • Usually I preferred percentages for measurements because I found it easier in terms of responsiveness

  • I used a margin of 5% for the page

  • For the colors I visited htmlcolorcodes.com, I find it very useful and quick and you can choose the value you prefer, HEX (mine!), RGB or HSL

  • Text fonts have been taken from Google Fonts linking the code in link element in HTML head. Sample textes are in italian ;)

  • The nav buttons have been taken from getcssscan.com. Button name is "Button 30 by Algolia" but I did some modification

  • I set a default font-size of 16px and a font-weight of 200. However h2 font-weight is set to 400
